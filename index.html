Gudacas Litsong Manok Inventory

Run the inventory app on your PC or phone. Download the source code, or follow the phone instructions below.

Open GitHub Repo Download ZIP

How to use

PC

Download the ZIP, extract it, then run:

python main.py

Android

Install PyDroid3 from Google Play, then open main_mobile.py in PyDroid3 and run it.

Notes

This is a Python app, not a browser website. The GitHub page gives you the source code and downloads. If you want a clickable phone installer, we still need to build an APK.
