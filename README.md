# Gudacas Litsong Manok Inventory System

A simple inventory management system with GUI for Gudacas Litsong Manok (Chicken Feet) product.

## Usage

Run the script with Python:

```bash
python main.py
```

The GUI will open with the Gudaca logo (if gudaca.jpg is present), input fields to add/remove products, and buttons to view inventory and get total value.

## Requirements

- Python 3.x with Tkinter (usually included)

## Features

- Add products with quantity, price, and automatic date
- Remove products by selecting in table with confirmation
- Search products in real-time
- Automatic inventory view updates with centered columns
- Table shows Name, Quantity, Unit Price, Total Price, Date
- Save inventory report to PDF or Excel with custom filename and location
